1. Из общего проекта все методы вызываются через сервис AndroidScreenCaptureService.
2. Для начала при запуске приложения пользователь должен дать разрешения в через методы CheckOverlayPermission() CheckNotificationPermission()
	- Разрешение на overlay нужно для плавающей панели.
	- Разрешение на notification нужно исплючительно для перемотки мудиа потока из нашей плавающей панели.
