## Ключевые особенности

* **Захват системного аудио:** Запись чистого звукового потока напрямую из операционной системы Android или любого приложения.
* **Плавающий виджет (Overlay):** Компактная оверлей-панель управления, которая отображается поверх других приложений для быстрого старта/остановки записи, перемотки воспроизведения.
* **Визуальный редактор:** Интуитивный интерфейс для редактирования аудио.
* **Экспорт** аудиофайлов в форматах `WAV` и `MP3`.

## Технологический стек

* **Фреймворк:** C# / .NET / Avalonia UI (кросс-платформенный интерфейс с кастомными анимациями).
* **Аудио-движок:** BASS
